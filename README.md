
Веб-приложение для отдела продаж: расчёт стоимости гофрокоробки с коэффициентами
наценки по объёму заказа, схемой коробки, экспортом КП в PDF и техкарты в Excel.
Работает на телефоне и компьютере, не требует сервера и аутентификации,
хранит настройки в браузере (localStorage).
Что внутри
Готовое приложение одним файлом (шрифт встроен). Просто откройте в браузере.
`index.html` + `app.js` + `fonts.js`	Разделённая версия (то же самое, но удобнее редактировать).
Для работы нужен интернет только при экспорте (подгружаются библиотеки jsPDF и SheetJS
с CDN). Сам расчёт и схема работают офлайн.
Возможности
Ввод длины, ширины, высоты (мм) с валидацией минимальных размеров
(высота ≥ 245, ширина ≥ 80, ширина + высота ≥ 320 — настраивается).
Схема коробки в изометрии с корректными пропорциями и подписями сторон.
Типы развёртки с разными формулами площади (FEFCO 0201 по техкарте,
FEFCO 0201 по формуле ТЗ, FEFCO 0200, 0203, лоток 0427).
Коэффициенты наценки по градациям объёма (1000+, 500–999, 300–499, 100–299, 50–99),
редактируются на странице «Настройки».
Скидка клиента в % (по умолчанию 0).
Цена 1 коробки без НДС, НДС, цена с НДС; сумма всего заказа без и с НДС.
Внутренний объём в литрах.
Вес одной коробки и партии (кг) для транспортного блока.
Экспорт PDF — бланк коммерческого предложения с логотипом (кириллица корректная,
шрифт PT Sans встроен).
Экспорт Excel — технологическая карта по шаблону вашего файла (все производственные
формулы раскроя + коммерческий расчёт).
Формула расчёта цены
    Цена 1 шт (без НДС) = Площадь заготовки (м²) × Цена картона за м² × Коэффициент наценки × (1 − Скидка)
    Цена 1 шт (с НДС)   = Цена без НДС × (1 + НДС/100)

Цена картона за м², плотность картона (для веса), ставка НДС и припуски задаются
в «Настройках».
Формула площади FEFCO 0201 (из вашей техкарты)
    Клапан            = Ширина / 2 + 4
    Длина заготовки   = 66 + Длина×2 + Ширина×2
    Ширина заготовки  = Высота + Клапан×2
    Площадь (м²)      = (Длина заготовки / 1000) × (Ширина заготовки / 1000)

Откройте вкладку Настройки в приложении:
отредактируйте коэффициенты наценки;
укажите реальную цену картона за м² (BYN) и плотность гофрокартона (г/м²);
впишите название компании и реквизиты — они попадут в шапку PDF-бланка;
при необходимости измените ставку НДС и ограничения размеров.
Настройки сохраняются в браузере. Кнопка «Сбросить к стандартным» возвращает
исходные значения.
